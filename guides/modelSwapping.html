Basic Model Swapping

Required resources

You'll need the following resources:


QuickBMS (RPKG extractor) QuickBMS script (required for QuickBMS to work) Patch Builder

Extracting the models you'd like to swap

First, find the two PRIM files you'd like to swap. A list of disguises and their associated PRIM files can be found here.


When you've found the IDs of the models you want to swap, extract them using the following command:


quickbms_4gb_files -f "*PRIM/[id of model, all lowercase]*" Hitman2_rpkg.bms "C:\Program Files (x86)\Steam\steamapps\common\HITMAN2\Runtime\chunk0.rpkg" "unpacked"

Ensure that Hitman2_rpkg.bms is located in the same folder as quickbms_4gb_files.exe . If you installed Hitman 2 in a different directory, adjust the command as needed.


Run the command twice, once for each of the models you'd like to swap.


Swapping the models

In the unpacked folder, there should be two .PRIM files. They may be in a sub-folder called PRIM.


Copy the PRIM file with the ID of the model you want the other model to be swapped TO. Paste it into a new folder.

Rename the file to the ID of the model you want swapped FROM.


For example, to replace the Phantom Suit's driving gloves (1c437ff0448691.PRIM) with the Signature Suit's normal gloves (a92c546029c638.PRIM), copy a92c546029c638.PRIM to a new folder and rename it to 1c437ff0448691.PRIM .


Packaging the mod

Drag your new folder (with the PRIM file) onto ZHM5PatchBuilder.exe and an RPKG file should be created. You can then load that RPKG file with QuickMod or manual naming and you're done!
